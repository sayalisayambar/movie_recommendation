# 🎬 Movie Recommendation System

An End-to-End Movie Recommendation System built using NLP, TF-IDF, FastAPI, and Streamlit.

## 🚀 Live Demo
https://movierecommendation-jbzkb3nxu7h8fke9fwvvar.streamlit.app/

## 💻 GitHub Repository
https://github.com/sayalisayambar/movie_recommendation

## ✨ Features
- Trending movies
- Movie search
- NLP-based recommendations
- TMDB API integration
- FastAPI backend
- Streamlit frontend

## 🛠️ Tech Stack
- Python
- NLP
- TF-IDF
- Scikit-learn
- FastAPI
- Streamlit
- TMDB API
